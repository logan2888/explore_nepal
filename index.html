About This Project

About ExploreNepal

Final Year Project — Phoenix College of Management, Kathmandu

What is ExploreNepal?

ExploreNepal is a comprehensive web-based tourism guide for Nepal, developed as a Final Year Project for the Bachelor of Information Technology program at Phoenix College of Management. The platform is built using HTML5, CSS3 and JavaScript for the frontend, with a Node.js and Express.js backend server and a MongoDB Atlas cloud database.

Why was it built?

Despite Nepal being one of the world's finest tourism destinations, its digital infrastructure is scattered and incomplete. Tourists struggle to find reliable up-to-date information in one place. ExploreNepal bridges this gap with a single unified platform covering 30 destinations, hotels, restaurants, maps and trip planning tools.
